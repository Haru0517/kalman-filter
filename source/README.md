## Requirements
- Python:3.7


## Usage
Install library:
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

Run:
```bash
python least_squares.py
```

The output graph images are in `graph` folder.
The log file is output to the `log` folder.
